About



  • Click inside the board to start the game.
  • This is a two player game.
  • Players can use "A" "D" keys and navigation keys to move the board.
  • You can use the buttons also.
  • You can get a point if opposite player miss the ball.
  • Once a player gets more than 5 points, the game ends.
